# Ontology Designer

本项目是一个“本体模型 + AI 驱动”方法论与可运行 MVP 的实践仓库，目标是用结构化本体模型（OBJ / BHV / EVT / RULE 等）支撑业务建模、校验、可视化与后续代码生成。

## 仓库结构

- `designer/`：Vue 3 + TypeScript 的本体模型设计器 MVP
- 根目录文档：方法论、规格书、场景需求、测试报告与使用手册

## 核心能力

- 四类核心模型建模：对象、行为、事件、规则
- 事件链可视化与聚合图可视化
- 一致性校验（悬空引用、环路、孤立事件等）
- YAML 导入导出（可作为 AI 生成输入契约）
- 示例场景：合同管理系统闭环

## 快速开始

### 1) 安装依赖

```bash
cd designer
npm install
```

### 2) 启动开发环境

```bash
npm run dev
```

### 3) 生产构建

```bash
npm run build
```
